Rahul Bethi




Objective:

  • The objective of the game is to escape from the enemy which will be trying to catch the player.
  • The main objective in making this game is to learn Ray Casting, Character Controller and integrating GUI elements like gun croohair.
  • Enemy will try to goto to the last known position of the player if it looses sight.


How To Play:

  • Arrow keys to move th player accordingly.
  • Free look with the help of mouse.
  • Hold 'Shift' key to run while moving forward.
  • Hold 'Tab' key to change to world camera.


Logic Design and Implementation:

  • Create two character controllers for player and enemy with collision detection.
  • Implement gravity logic so the player and enemy must be on the ground.
  • Impliment Raycast for the player, so that the gun will always point the end point of the Ray.
  • Implement Raycast between enemy and player, so that enemy can see if the player is in line of sight.
  • Implement logic for the enemy to gotowards the player or goto the last known position of the player.
  • Implement running of the player when holding shift key.




Scope for improvement:

  • Implement bullet animation to kill enemy.
  • Implement logic for the enemy to climb up the building.
  • Implement Audio to the game.
