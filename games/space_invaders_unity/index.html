Author:

Rahul Bethi




Objective:

  • The objective of the game is to shoot the aliens aproaching downwards, and dodge the bullets they shoot at you.
  • The Player ship gets destroyedwhen it is struck by the enemy bullets or when it goes into enemy ships.
  • Main objective is to learn how to use prefabs and particle systems. And also triggers and collisions, all in Unity 3D.


How To Play:

  • Use arrow keys as joystick keys to move the player ship around.
  • Press 'Space' key to fire bullets.


Logic Design and Implementation:

  • Create enemy class, player bullet class, enemy bullet class and make prefeabs of those.
  • Create enemy and bullets' movement logics.
  • Create movement logic for player, such that on pressing the required keys or the combination of those keys, vertical, horizontal or diagonal movement can be obtained, like joystick movement.
  • Efficiently use bullets and enemies with their respective prefabs by deactivatng and activating them.
  • Particle systems are used to show destruction.
  • Create variables to hold the score, number of lives of the player.
  • Create logics to determine when the game is over and display it on the screen.




Scope for improvement:

  • Use real models in place of squares.
  • Use UV scrolling for the background.
  • Implement Audio to the game.
