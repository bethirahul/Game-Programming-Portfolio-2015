Objective:

  • The objective of the game is to acquire the Power-ups by moving the Snake and reach the score of 1000 first.
  • The Snake can be moved in 8 directions using single/multiple arrow keys.
  • The speed and length of the Snake varies depending upon the powerups acquired.
  • The Snake dies if it hits the wall or another Snake.
  • If both the heads of the Snakes hit eachother, then both loose.


How To Play:

  • Arrow keys, to move the Left Snake.
  • W S A D keys, to move the Right Snake.


Logic Design and Implementation:

  • Create Snake class, which comprises of an array of square objects(nodes) to constitute the Snake body.
  • Create another two arrays to store x and y coordinates such that they keep track of each pixel of each square node constituting the Snake body, so that using these arrays, the snake can be moved at any desired speed.
  • Create movement logic, such that on pressing the required keys or the combination of those keys, vertical, horizontal or diagonal movement can be obtained.
  • Only the head of the snake (first node) reacts to the input creating n last positions while it moves.
  • Create logic so that the snake cannot go backwards onto itself by checking the position of the second node.
  • Each node will moving to the next node's 'n'th last position. 'n' depends on the size of each node.
  • Create variables to hold the score, speed and number of nodes of a snake.
  • Create logics to determine when the game is over and display it on the screen.




Scope for improvement:

  • Implement more varieties of power-ups like making the snake immune for some time using a timer.
  • Implement logic for the snake to go and come from one side to the other side like teleporting.
  • Implement Audio to the game.
