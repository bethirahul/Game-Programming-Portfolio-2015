Objective:

  • The objective of the game is to arrange all tiles in an order by moving them.
  • The tile which is adjacent to the empy space can be moved into the empty space.
  • Timer will start counting as soon as the game starts, so to count the time taken to think into consideration.
  • Tiles can only move horizontally or vertically, but not in diagonal.


How To Play:

  • Click a tile which is adjacent to the empty space, for the tile to move into the empty space.


Logic Design and Implementation:

  • Create 16 tiles as Objects and leave the 16th tile as a blank tile.
  • Move the tiles in random many times to scramble. This method ensures that the pattern is solvable.
  • Create movement logic, such that on pressing the tile beside the empty space gets swaped with the empty space.
  • Create variables to hold the time taken and the number of moves made to solve the game.
  • Create logics to determine when the game is over and display it on the screen.




Scope for improvement:

  • Use square pieces of a picture instead of numbers on the tiles.
  • Implement Audio to the game.
