Objective:

  • The objective of the game is to collect the shimmering challice and reach the door.
  • Dave can move forwards and backwards using the arrow keys, he can jump and fall with an acceleration due to gravity.
  • Dave can shoot bullets in the direction he faces. He has to shoot the monster which comes in his way.
  • Dave dies, when struck by monster bullets, or when he doesnt land up on a solid tile, or when he bumps into the monster.
  • The game completes when Dave reaches the door.
  • Dave can collect Jewels of different colors to get different scores.


How To Play:

  • Move left and right using the 'Left' and 'Right' arrow keys.
  • Jump using the 'Up' arrow key.
  • Shoot using the 'Alt' key, once Dave acquires a gun.


Logic Design and Implementation:

  • Create Level class, which comprises of a Bitmap, each number represents the position of different objects (red bricks, weed, gun, door, challice, fire, water, jewels (blue and red diamonds, purple stone)) in the level.
  • Use Sprites to create movement, jumping and death animations of all animating objects (Dave, weed, fire, water, enemy, enemy-bullet, challice).
  • Create logics for detecting collisions between dave and other objects around him by determining in which location he is in the Level Bitmap.
  • Create jumping logics in order to determine the jumping and falling speeds of Dave when he freely falls under gravity.
  • Scroll the environment according to the position of Dave.
  • Create variables to hold the score and the number of lives of Dave.
  • Create logics to determine when the game is over.




Scope for improvement:
